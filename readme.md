# 基于MFC的简易计算器

#### 简单计算器  基本要求：

 (1) 基本软件界面设计与绘制生成； 

(2) 程序能实现：加，减，乘，除，乘方，开方，倒数，三角函数等如图所示所有运 算功能；

 (3) 实现数据的输入，输出，运算，显示及程序退出等功能;

(4) 具有输入数据的错误检查功能；

 (5) 实现科学型和标准型计算器界面转换。
